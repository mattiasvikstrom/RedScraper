Redscraper

This flaskapplication allows for scraping of subreddits. Both posts and comments can be scraped seperatly. The pages analyze section allows for analytical scripts to be run and provide you with valuable insights

Getting started

Head over to the scape section of the site, there options to select type of data to acquire and from which subreddit. The scraper will acquire the maximum allowed and avaliable data within the limitations of the praw api. Scraping comments will result in a larger dataset than topics, which is to no surprise that the time it will take is substantionaly longer. When the scrape is complete the file with data will be avaliable in the analyze section, there most mentioned and sentiment analysis can be performed and the results will be graphicaly displayed.
